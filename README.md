## 기능 요구사항

### [도메인 관점에서 기능 정리]

로또 (Lotto)
1. 번호의 숫자 범위는 1~45이다.
2. 1개의 로또를 발행할때 중복되지 않는 6개 숫자 뽑는다.
3. 가격 1장 = 1000원

당첨 (Won)
1. 번호 추첨시 중복되지 않는 수 6개 + 보너스 수 1개
2. 1등부터 5등까지 존재
(세부 조건 생략)

구입한 사람(Person)
1. 구매한 로또 개수 = 구입 금액 / 로또 1장 가격


### [절차 관점에서 기능 정리]

1.당첨 번호, 보너스 번호 입력
( 쉼표 기준으로 구분 )
( 예외 상황에서 에러 문구 출력 )

2.발행한 로또 수량 및 번호 출력
( 오름차순 정렬 )

3.당첨 내역 출력

4.총 수익률 출력
( 소수점 둘째 자리에서 반올림 )
